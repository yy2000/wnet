
Elastic Net with weighted errors

Setup
=======================

conda
~~~~
conda create -n kz python=3.6
source activate kz 
pip install numpy
~~~~

install with 
~~~~
git clone --recursive https://github.com/bodono/scs-python.git
cd scs-python
python setup.py install
~~~~

Run it
==========================
~~~~
python weighted_enet.py
~~~~
